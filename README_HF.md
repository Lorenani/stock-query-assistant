# 📈 智能股票查询助手

一个基于大语言模型的交互式股票查询与分析系统。

## ✨ 功能特色

- 🤖 **AI驱动的智能对话**：基于通义千问大模型
- 📊 **多维度数据分析**：历史价格查询、多股票对比
- 🔮 **时间序列预测**：ARIMA模型预测股票价格
- 📰 **实时新闻搜索**：获取股票热点新闻
- 🎨 **现代化Web界面**：美观流畅的用户体验

## 🚀 快速开始

1. 在右侧输入框中输入您的问题
2. 点击提示按钮快速开始
3. 查看分析结果和可视化图表

## 💡 使用示例

- 查询2024年全年贵州茅台的收盘价走势
- 对比2024年中芯国际和贵州茅台的涨跌幅
- 预测贵州茅台未来7天的收盘价
- 贵州茅台最近的热点新闻

## 🔧 技术栈

- Python 3.11+
- Qwen Agent (通义千问)
- ARIMA时间序列分析
- Matplotlib数据可视化

## 📝 注意事项

本项目需要配置以下环境变量：
- `DASHSCOPE_API_KEY`: 通义千问API密钥
